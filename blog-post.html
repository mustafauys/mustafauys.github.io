Flutter ile Mobil Uygulama Geliştirme

25 Aralık 2023 | Yazan: Mustafa Uysal

Flutter Nedir?

Flutter, Google tarafından geliştirilen açık kaynaklı bir UI yazılım geliştirme kitidir. Tek bir kod tabanından iOS ve Android platformları için yüksek performanslı, yüksek kaliteli native uygulamalar geliştirmenize olanak sağlar.

Neden Flutter?

Flutter'ın en önemli özelliklerinden biri, "hot reload" özelliği sayesinde yapılan değişikliklerin anında görüntülenebilmesidir. Bu özellik, geliştirme sürecini önemli ölçüde hızlandırır ve verimliliği artırır.

Flutter ile Uygulama Geliştirme Süreci

Flutter ile uygulama geliştirme sürecinde, Dart programlama dilini kullanırız. Dart, object-oriented bir programlama dili olup, Java ve JavaScript'e benzer bir syntax'a sahiptir. Flutter'ın widget-based mimarisi, UI geliştirmeyi oldukça kolaylaştırır ve zengin bir widget kütüphanesi sunar.

Örnek Bir Flutter Uygulaması

Basit bir Flutter uygulaması geliştirerek başlayabilirsiniz. İşte temel bir Flutter uygulaması örneği:


                    import 'package:flutter/material.dart';

                    void main() {
                    runApp(MyApp());
                    }

                    class MyApp extends StatelessWidget {
                    @override
                    Widget build(BuildContext context) {
                        return MaterialApp(
                        home: Scaffold(
                            appBar: AppBar(
                            title: Text('İlk Flutter Uygulamam'),
                            ),
                            body: Center(
                            child: Text('Merhaba Flutter!'),
                            ),
                        ),
                        );
                    }
                    }
